# AlphaGBM 波动率曲面

## 前置条件

- **API Key**：设置环境变量 `ALPHAGBM_API_KEY`（格式 `agbm_xxxx...`）。
- **Base URL**：默认 `https://alphagbm.zeabur.app`。可通过环境变量 `ALPHAGBM_BASE_URL` 覆盖。

## 功能说明

为任何可期权标的构建 **3D 波动率曲面**，在两个维度上映射隐含波动率：行权价（虚实程度）
和到期时间。识别曲面上哪些期权相对于拟合曲面偏贵、偏便宜或存在异常。

### 核心输出

| 输出 | 说明 |
|------|------|
| **曲面网格** | 每个（行权价，到期日）坐标的 IV 值，完整3D映射 |
| **ATM 期限结构** | 平值 IV 跨不同到期日的变化（近月 vs 远月）|
| **各到期日偏斜** | 每个到期日的 Put/Call IV 差值，衡量恐慌/自满程度 |
| **曲面异常点** | IV 显著偏离拟合曲面的点，潜在错误定价机会 |
| **曲面形态** | 分类：正价差 / 反价差 / 平坦 / 倒置 / 事件驱动 |

### 曲面告诉你什么

- **正价差（Contango）**（近月IV < 远月IV）：正常市场，近期无恐慌
- **反价差（Backwardation）**（近月IV > 远月IV）：预期有近期事件（财报、FDA 等）
- **陡峭偏斜**：市场在给 Put 定尾部风险溢价，对冲需求强
- **平坦偏斜**：情绪均衡，无明显方向性恐慌
- **检测到异常**：某个合约相对邻近点定价异常，潜在机会

### 波动率风险溢价（VRP）

```
VRP = 隐含波动率 - 历史波动率
```

| VRP 等级 | 卖方 | 买方 |
|----------|------|------|
| very_high（极高，≥15%）| 非常有利 | 不利 |
| high（偏高，5-15%）| 有利 | 略不利 |
| normal（正常，±5%）| 中性 | 中性 |
| low（偏低，-15% 到 -5%）| 不利 | 有利 |
| very_low（极低，<-15%）| 非常不利 | 非常有利 |

## API 端点

### 波动率曲面（3D）

```
GET /api/options/tools/vol-surface/<SYMBOL>
```

返回完整3D波动率曲面，包含虚实程度轴、到期日轴和 IV 网格。

### IV 快照（快速检查，无额度消耗）

快速检查平值 IV / IV Rank / 历史波动率 / VRP，不拉取完整曲面：

```
GET /api/options/snapshot/<SYMBOL>
```

返回：平值IV、IV Rank、30日历史波动率、VRP、VRP等级。

## 使用方法

### 输入
- **必填**：股票代码
- **可选**：虚实程度范围（如 0.8-1.2）、到期日范围（如 7-90 天）
