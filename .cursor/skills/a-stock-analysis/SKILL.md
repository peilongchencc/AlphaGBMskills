# A股单只股票综合分析

## 概述

对单只 A 股进行多维度量化分析，最终输出 **风险评分（0-10）** 和投资建议。

## 分析维度

### 1. 基本面评估

| 指标 | 评分规则 |
|------|----------|
| PE（TTM） | > 60: +2分，30-60: +1分，< 30: 0分 |
| PB | > 8: +1分，< 1: -1分（可能低估） |
| ROE | < 8%: +1分（盈利能力弱） |
| 营收增长率 | < 0: +2分，0-10%: +1分 |

### 2. 技术面评估

| 指标 | 评分规则 |
|------|----------|
| 股价 vs MA250 | 低于 MA250: +1分（长期趋势向下） |
| RSI(14) | > 80: +1分（超买），< 20: +1分（超卖恐慌） |
| MACD | 死叉且柱状缩短: +1分 |
| 布林带 | 触及上轨: +0.5分 |

### 3. 资金面评估

| 指标 | 评分规则 |
|------|----------|
| 换手率 | > 15%: +1分（过度投机） |
| 量价背离 | 价升量缩: +1分 |

## 风险等级划分

- **0-2 分**：低风险，可考虑建仓
- **3-5 分**：中等风险，谨慎操作
- **6-8 分**：高风险，建议观望
- **9-10 分**：极高风险，远离

## 使用方式

```bash
cd scripts
python stock_analysis.py
```
