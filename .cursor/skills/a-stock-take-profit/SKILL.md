# A股止盈策略分析

## 概述

本技能提供 A 股止盈策略的量化分析能力，核心指标为 **过山车率（Rollercoaster Rate）**：
衡量"已经赚到手的利润又回吐了多少"。

## 核心概念

### 过山车率

> 过山车率 = 从峰值回撤超过阈值的次数 / 总入场次数

- 入场条件：每月第一个交易日买入
- 判定逻辑：浮盈达到 X% 后，又从峰值回撤超过 50%（即利润腰斩）
- 意义：过山车率越高，说明不止盈的代价越大

### 三种止盈策略

| 策略 | 说明 | 参数 |
|------|------|------|
| 固定止盈 | 涨到目标位直接卖出 | target_pct: 20%/30%/50% |
| 移动止盈 | 从最高点回撤超过阈值时卖出 | trail_pct: 10%/15%/20% |
| 分批止盈 | 分 3 档逐步减仓 | levels: [20%, 35%, 50%] |

## 使用方式

```bash
cd scripts
python take_profit.py
```
