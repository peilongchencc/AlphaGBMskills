# A股多股横向对比

## 概述

将多只股票放在一起进行量化对比，找出当前阶段最优标的。

## 对比维度

| 维度 | 指标 | 排名方式 |
|------|------|----------|
| 估值 | PE(TTM)、PB | 越低越好 |
| 成长性 | 营收增长率、ROE | 越高越好 |
| 趋势 | 近60日涨跌幅、MA20偏离度 | 视情况 |
| 波动率 | 年化波动率 | 越低越好（稳健型） |
| 收益风险比 | 夏普比率（近1年） | 越高越好 |

## 输出格式

对比矩阵 + 综合排名 + 各维度最优标的

## 使用方式

```bash
cd scripts
python compare_stocks.py
```
